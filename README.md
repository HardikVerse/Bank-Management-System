# Bank Management System

A simple Bank Management System built using Python.

This project is part of my Python learning journey, where I explored Object-Oriented Programming (OOP), file handling, JSON data storage, input validation, and error handling by building a real-world console application.

## Features

* Create bank accounts
* Generate unique account numbers
* Account authentication using Account Number and PIN
* PIN validation
* Date of Birth validation
* Automatic age calculation
* Age restriction (18+ accounts only)
* Deposit money
* Withdraw money
* View account details
* Update account details
* Delete accounts
* Store user data using JSON
* Error handling for invalid or empty database files
* Menu-driven interface
* Object-oriented design using classes
