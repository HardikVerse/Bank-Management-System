# Bank Management System

A simple Bank Management System built using Python.
This project is part of my learning journey in Python and file handling.

## Features

* Create bank account
* PIN validation
* Age validation
* Store user data using JSON
* Error handling for invalid database files
* Menu-driven interface
* Object-oriented programming using classes

## Technologies Used

* Python
* JSON
* pathlib

## Project Structure

```bash
main.py
users_data.json
```

## How to Run

1. Clone this repository

```bash
git clone <your-github-repo-link>
```

2. Open project folder

```bash
cd bank-management-system
```

3. Run the program

```bash
python main.py
```
